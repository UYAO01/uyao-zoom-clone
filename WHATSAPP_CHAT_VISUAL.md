Click + Drag LEFT\n  Right swipe: Click + Drag RIGHT\n  Menu: Click ⋮ button\n  Reply: Type in input box\n\nAll actions work exactly like mobile!\n```\n\n### Mobile (Touch)\n```\nMobile:\n  Left swipe: Touch + Swipe LEFT\n  Right swipe: Touch + Swipe RIGHT\n  Menu: Tap ⋮ button\n  Reply: Type in input box\n\nOptimized for touch screens!\n```\n\n---\n\n## Swipe Direction Reference\n\n```\nYour
